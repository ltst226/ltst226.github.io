Power Plants Across the U.S.

This map was created in MAP 672: Programming for Web Mapping. This course was developed to help students in an introduction for web programming and javascript coding. These programs were written within a program called Brackets which was used alongside Github to store and present the data and maps. The final project of this class is the web portfolio you are reading now.

This proportional symbol map was created with data gathered from the Energy Information Administration. The point data gathered from the EIA was stored in a file and was added to the programming we wrote. We were then able to take that point data, change it, style it, and make the map interactive. All of the information we learned throughout this course was added to this map.

  • See the map: Power Plants Across the U.S.
  • Data Source: Energy Information Administration

Ohio Unemployment Rates

This map was created in MAP 673: Design for Interactive Web Mapping. This course was developed to help students in an introduction for web programming and javascript coding. These programs were written within a program called Brackets which was used alongside Github to store and present the data and maps.

This choropleth symbol map was created with data gathered from Ohio's Labor Market Information. I took the data and created a .CSV file for every county in Ohio for 8 years. Through coding we learned during that lesson, I was able to take that data, create a choropleth map and have the years change with the data slider. I really enjoyed making this map and seeing how different parts of Ohio was affected when the stock market and housing market bubble busted in 2008.

  • See the map: Ohio Unemployment Rates from 2005 to 2013
  • Data Source: Ohio Labor Market Information

Reported Crimes on University of Kentucky

This map was created in MAP 673: Design for Interactive Web Mapping. This course was developed to help students in an introduction for web programming and javascript coding. These programs were written within a program called Brackets which was used alongside Github to store and present the data and maps.

This map was created with data gathered from the University of Kentucky's Crime Log. The data covers from January 2013 to the beginning of June 2016. The point data was gathered in a .CSV file and uploaded to CartoDB. I then took the data and converted it to a GeoJSON format and styled the format and data within Brackets. I took the data and made it interactive with an interactive propmt and a date slider. I also created a heat layer to represent "hot" spots of crime on campus.

  • See the map: Reported Crimes Across University of Kentucky.
  • Data Source: University of Kentucky Crime Log
